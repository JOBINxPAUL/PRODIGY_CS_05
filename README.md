# Network Packet Analyzer
This Python program captures and analyzes network packets, displaying relevant information such as source and destination IP addresses, protocols, and payload data. Ensure the ethical use of this tool for educational purposes only.

# Features
Packet Capture: Captures live network packets on a specified interface.
Protocol Analysis: Identifies and displays information about TCP and UDP protocols.
Payload Display: Shows the payload data of captured packets.

# Requirements
Python 3.x
scapy library

# Ethical Considerations
Network packet analyzers can capture sensitive information, including private communications and credentials. Use this program responsibly and only with explicit consent from the network owner. Ensure compliance with all relevant laws and regulations.
